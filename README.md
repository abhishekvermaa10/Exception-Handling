<h1 align="center">Exception Handling</h1>

<p align="justify">
    Welcome to the <b>Exception Handling</b> repository! This repository provides hands-on demos covering the
    fundamentals of handling exceptions in Java. It explores different exception-handling mechanisms, including
    try-catch blocks, multi-catch, propagation, custom exceptions, and the proper use of <b>throw</b> and <b>throws</b>.
    By mastering these techniques, you can write more robust and fault-tolerant applications. </p>

<hr>

<h3 align="left">üìå Table of Contents</h3>
<ol align="left">
    <li>Scenario without Exception Handling</li>
    <li>Handling exceptions and Understanding various ways to show messages in catch block</li>
    <li>Handling multiple exceptions using multiple catch blocks and multi-catch block</li>
    <li>Propagating exception to calling class</li>
    <li>Understanding throw keyword</li>
    <li>Understanding throws keyword</li>
    <li>Understanding try-catch-finally</li>
    <li>Utilizing try-with-resources</li>
    <li>Creating and handling custom exception</li>
</ol>
